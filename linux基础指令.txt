目录相关

	创建：
		mkdir -p ./d1/d2/1.txt  ＃　可以创建多级目录
		、
	
	删除：
		rm -rf d1
		：


	移动：
		mv -r d2 d1
		一般在移动前最好先备份目录，例如：
			cp -r d d.bak


	复制：
		cp -r d2 d1		
		-p  ＃　保留文件属性,preserve	
		-a  ＃　用于实现备份
		一般用户复制多个文件或目录到一个目录,若是目录需要加上 -r 选项：
			cp /bin/{zdiff,zegrep,zless} d1

	
	查询：
		find -type d -name '*2'  
			# 在当前目录以及子目录下查询名称以２结尾的目录
	

	查看：
	    ls -l   
	    	查看当前目录下所有的子目录与文件的详细信息，主要内容如下：
	        第一位：
	            _：普通文件
	            d:目录
	            l：链接文件
	        第２－１０位：用户权限
	            属主权限：r读权限，w写权限，x执行权限
	            属组权限
	        
	        文件大小
	        
	        时间戳

	 	cd：进入目录
	 	
	 	pwd：查看当前的绝对路径

	 	tree：查看目录树



文件相关：

	新建：
		touch f1.txt
		、


	修改：
		touch -a 1.txt：更新文件的access时间
		touch -m 1.txt：更新文件的modify时间
		
	移动：
		mv ./d1/1.txt ./d2	
		移动前最好备份:
			cp d1/1.txt d1/1.txt.bak

	删除：
		rm f1.txt		
		

	复制：
		cp ./d1/1.txt ./d2
		若文件名重名，则直接覆盖文件。
		

	查询：
		-type d　　＃　表示目录
		-type f　　＃　表示文件
		find -type f -name '*.txt'  # 在当前目录以及子目录下查询名称以２结尾的文件

	查看：

		stat：查看文件所有属性

漏斗家族：
	将正确的命令结果输入到文件中：
		>>	追加重定向，追加到最后一行
		>	标准重定向，先清空，再写入内容

	只将错误的命令结果输入到文件中：
		2>> 

	不论命令错误与否都输入到文件中：
		commands >>3.txt 2>>3.txt


	
文件内容相关：

	新增内容：

		vim
			:wq!	强制退出并保存
			:q!		强制退出
			G       跳转到最后一行
			gg      跳转到第一行
			i       进入编辑模式
			o       增加下一空白行并进入编辑模式
		    esc     退出编辑模式
		    dd      删除本行
		    ０      行首
		    $       行尾
		    #x      删除光标后的x个字符
		    ctrl+f  向后翻屏
		    ctrl+b  向前翻屏
		    yy      复制当前行
		    p       粘贴至下一行
		    P       粘贴只上一行
		    u       撤销前一次编辑操作
		    ctrl+r  撤销最近一次的撤销操作
		
		echo       
		         
			追加一行内容到文件末尾：
				echo asasas >> file

			清空原文件并写入内容：
				echo asasas > file

			追加多行内容：
				cat >> 3.txt << EOF
				> q
				> w
				> e
				> rt
				> EOF

			
	查看内容：

		cat:
		    一般查看所有内容；
		    -n ：显示行号
		    -c  按字节显示，例如-c3显示前三个字节
		    --help 查看介绍,和ｍａｎ作用类似，但包含中文

		
		head：
		    默认查看前１０行内容
		    -n      显示前ｎ行   

		            	
		tail：
		    默认查看后１０行内容
		    -n      显示最后的ｎ行

		    
		sed:
			显示某一行：			    
				sed -n '3p' 3.txt

			显示多行：
				sed -n '2,6p' 3.txt　　＃　显示２－６行
			
			不显示　或　删除　匹配的行：
				sed '/p/d' 3.txt  ＃　不显示包含字符p的行
				-i  ＃　直接删除

			替换内容：
				sed 's#a#b#g' 1.txt -i  ＃ 将出现的字符ａ替换成字符ｂ

	
		awk:
			内容比较多！
			显示某一行或多行：
				awk 'NR==n' 3.txt  ＃显示第ｎ行内容
				awk "NR==n, NR==m" 3.txt　　＃　显示ｎ－ｍ行的内容   

		
		grep：
		     grep 'ab' 1.txt    ＃　查看包含特定字符的所有行
		     grep -v '2' 1.txt  ＃　列出所有不包含字符２的行
		
	
		｜管道符：
			处理管道符之前的内容
				head -10 3.txt | tail 3  ＃　取８－１０行的内容 


		cut：
			-d：指定分隔符，默认是一个空格
			-f：指定要显示的字段

				例如：按照空格分割，显示每行的第一个字段
				fangqihan@fangqihan-PC:~/data$ cat 1.txt
					1 12 asdasda
					a 12as asdasdv
					vc 12as asdasds
					d 12a asdasdf
				fangqihan@fangqihan-PC:~/data$ cut -d' ' -f1 1.txt
					1
					a
					vc
					d

				cut -d' ' -f1,2 1.txt  ＃　显示第一个和第二个字段
				cut -d' ' -f1-3 1.txt  ＃　显示第１－３个字段


		sort：
			查看文件内容并按行排序；
			默认按照行首个字符在ascii对应的十进制升序排序。

			-n  按照数值大小排序
			-r  降序
			-t  指定分隔符
			-k  指定字段
				sort -t: -k3 -n /ect/passwd
			-u  重复行只显示一次
			-f  不区分大小写


		uniq:
			相邻并完全一样的行只显示一次
		
			-d  显示重复行，只显示一次
			-D  显示重复行，有多少显示多少
			-c  显示重复的行和对应的次数


		wc:
			统计文本内容（行，单词，字节数）
			fangqihan@fangqihan-PC:~/data$ wc 1.txt
				10 　18 　80 　1.txt
				行　　单词　字节　

			-l  显示行数
			-w  显示单词数
			-c  显示字节数
			-m  字符数
			-L  最长的一行包含的字符数


		tr :
			替换字符,其中ac代指集合，而非一个字符串：
				tr 'ac' 'bd' < file  将文件中的字符ａ替换成字符ｂ
				tr 'a-z' 'A-Z' < file  将小写换成大写字母

			-d 　删除指定的字符
				tr -d '1' < file

	命令历史history：	
		用户命令历史会缓存在文件中；
		-c  清空命令历史
		-d600  删除第600个命令
		-d600 10 指定偏移量１０，删除第６００－６１１行命令

		!n 　　打印命令历史第ｎ条记录并显示执行结果
		!-n  　倒数第ｎ条命令
		!!	 　执行上一条命令
		！string   执行命令历史中最近一个以指定字符串开头的命令
